Accessibility

Use the accessibility button (top-right) to open the panel. All settings are saved in localStorage under accessify-settings.

Features: Text size (80%–200%), Contrast (Normal / High / Dark), Spacing (Normal / Wide), Font (Default / Dyslexia), Links (highlight on/off), Cursor (highlight circle on/off), Reset.

Features
  • Text size — Decrease or increase page text (80%–200%).
  • Contrast — Cycle Normal, High, and Dark modes.
  • Spacing — Toggle Normal or Wide text spacing.
  • Font — Switch between Default and Dyslexia-friendly font.
  • Links — Turn link highlighting on or off.
  • Cursor — Show a focus circle that follows the pointer.
  • Reset — Restore all settings to defaults.
